- Para Iniciar o Cliente e o Servidor, favor modificar p/ inicialização multipla
os projetos CLIENT e SERVER.

ex: /p lucas oi
( mensagem direta)

/to lucas oi
(mensagem normal)

- Para simular varias pessaos no chat, Executar o Arquivo CLIENT.EXE na /bin/ do projeto Client.
